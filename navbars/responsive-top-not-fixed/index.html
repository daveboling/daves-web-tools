Responsive Top Navbar w/ SlickNav.js and Bootstrap

Tools used: 
1. HTML
2. CSS3
3. Bootstrap Grid
4. jQuery
5. ...SlickNav.js

There is a fixed class that is not activated for personal usage & and scrolling.
Note: Scrolling-fixed has not been added.


Q: Why not use Bootstrap navbar?

A: While I greatly admire all of Bootstrap, I wanted to do my own work on this to prove that I could do it. Also, I don't want all my websites to have the 'Bootstrap look'. So, I merely use the grid system that it has provided, which is awesome.
